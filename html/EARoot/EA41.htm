[WMDS Category 3] An observing facility (station/platform) can be anything that supports making observations, e.g., a fixed station, moving equipment or a remote sensing platform. In abstract terms, an observing facility groups a near colocation of observing equipment managed by a single entity or several entities.<br/>Multiple WIGOS Station Identifiers can be declared as a comma-separated list using the attribute gml:identifier. By convention, the first entry is considered to be the primary WIGOS Station Identifier.<br/>The observing facility's name can be specified using the attribute gml:name. <br/>
